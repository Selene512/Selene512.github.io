# 企鹅 · AI 社交助手

> 一款运行在浏览器中的 AI 聊天辅助工具，帮你分析关系、生成回复、管理待办，让社交不再费脑。

---

## 功能概览

### 💬 智能回复生成
- 点击按钮，AI 根据当前对话上下文自动生成三条候选回复（最优、简短、有趣）
- 支持一键「用这条」将回复填入输入框，也可保存为话术备用

### 🤖 真实 AI 对话（需配置 API Key）
- 联系人回复由 Claude（Anthropic）驱动，模拟真实对话节奏
- **灵镜 ✦**：专属 AI 内心助手，可自由倾诉、梳理思路、进行自我反思

### 👤 人设系统
- 设置你的昵称、说话风格（活泼 / 温柔 / 酷 / 文艺 / 务实）
- 自定义口头禅、句子长度、表情使用频率
- 导入真实聊天记录，让推荐更贴合你本人的说话方式

### 📊 关系面板（右侧控制台）
每位联系人都有详细的关系档案：
- **亲密度评分**：好友 / 中等 / 疏远分层管理
- **话题标签**：掌握对方感兴趣的领域
- **暖场触发器**：智能提醒你何时主动出击
- **情感维度雷达**：幽默感、亲密度、深度等多维度分析
- **相处指南**：记录与对方的相处技巧和注意事项

### ✅ 智能待办
- 自动从对话中识别截止日期和任务（如"下周一提交"、"下午三点评审"）
- 支持紧急度分级（🔴 紧急 / 🟡 即将 / 🔵 普通）
- 内置日历视图，按日期筛选任务

### 💾 话术库
- 收藏满意的回复，下次直接复用
- 支持关键词搜索

## 快速开始

### 直接打开
本项目为纯前端单文件，无需任何构建工具：

```bash
# 直接用浏览器打开
open index.html
```

### 配置 API Key（解锁 AI 功能）
1. 点击顶栏右上角的「🔑 API Key」按钮
2. 输入你的 [Anthropic API Key](https://console.anthropic.com/)
3. 保存后，联系人回复和灵镜功能将由真实 AI 驱动

也可通过 URL 参数传入：
```
index.html?key=sk-ant-xxxxxx
```
